# :pushpin: Sistema de Cadastro de Produtos 
Projeto desenvolvido em Java com objetivo de treinar conceitos de programação e construir um sistema simples para gerenciamento e manipulação de produtos, incluindo validação de dados e atualização de preços.

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)

## Funcionalidades
- Cadastro de produtos: permiti inserir novos produtos no sistema com informações como código, nome e preço.
- Validação de código duplicado: impede a inclusão de produtos com código já existentes para garantir integridade dos dados.
- Aumento de preço por porcentagem: possibilita atualizar o preço dos produtos de forma automática.

## Conceitos utilizados 
- Java 17+ : linguagem principal do projeto
- POO (Programação Orientada a Objetos): encapsulamento, herança e métodos.
- Collection e Stream API: manipulação de listas, filtros e buscas eficientes.
- Validação de dados: regras de negócio aplicada para consistência e confiabilidade.
